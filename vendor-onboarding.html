¿Qué es neuralgpt.store?

neuralgpt.store es un marketplace exclusivo de software digital: apps, automatizaciones, herramientas de IA, scripts y soluciones técnicas. No vendemos hardware ni productos físicos. Conectamos desarrolladores y creadores de software con compradores técnicos hispanohablantes que buscan herramientas verificadas y de calidad.

🗄️ Modelo de hosting — tú alojas tu software

neuralgpt.store no almacena tu software. Tú alojas tu software en tu propia infraestructura y nos proporcionas una URL de descarga pública y estable. Nosotros validamos la URL, verificamos el software y gestionamos los pagos. Tras cada pago confirmado, redirigimos automáticamente al comprador a tu URL.

🐙
GITHUB RELEASES

La opción más popular. Sube tu .zip, .exe, .AppImage o APK a GitHub Releases. URL pública, gratuito, con versionado incluido.

☁️
CLOUDFLARE R2

Almacenamiento de objetos rápido y económico. Perfecto para archivos grandes. URL directa, sin egress costs.

🌐
SERVIDOR PROPIO

Si tienes servidor propio o VPS, aloja los archivos directamente. Control total sobre el hosting.

REQUISITOS DE LA URL DE DESCARGA
  • ✓ URL pública, accesible sin autenticación adicional
  • ✓ URL estable — no puede cambiar sin notificarnos con antelación
  • ✓ El archivo debe ser el exactamente descrito en la ficha del producto
  • ✓ HTTPS obligatorio
  • ✗ No se permiten URLs de servicios que expiren (WeTransfer, etc.)

🛡️ Proceso de verificación del software

Antes de publicar tu producto, lo verificamos para garantizar la seguridad de los compradores. El proceso tarda 24-48h laborables.

1
Análisis anti-malware

Ejecutamos tu archivo a través de VirusTotal y análisis interno adicional. Si detectamos cualquier comportamiento sospechoso, te notificamos y pausamos el proceso hasta resolverlo.

2
Verificación de la URL

Comprobamos que la URL de descarga es estable, accesible, el archivo se descarga correctamente y corresponde al producto descrito.

3
Revisión de la ficha

Verificamos que la descripción del producto, las plataformas compatibles y las funcionalidades declaradas son precisas y no engañosas.

4
Aprobación y publicación

Tras la aprobación, tu producto aparece en el marketplace con el badge "Software Verificado". Los pagos se activan automáticamente.
